# PLC Structured Text Draft Generator

## 1. 프로젝트 개요

본 프로젝트는 IO 맵과 사양서 텍스트를 기반으로  
IEC 61131-3 Structured Text(ST) 초안을 생성하기 위한 개인 개발 프로젝트입니다.

PLC 제어 로직 작성 과정에서 반복적으로 발생하는  
입력 정리, 데이터 구조화, 기본 검증, 코드 초안 생성 과정을 하나의 흐름으로 구성하는 것을 목표로 합니다.

생성 결과물은 실무 적용 전 검토가 필요한 초안(draft) 기준이며,  
엔지니어가 후속 수정 및 보완을 수행하는 것을 전제로 합니다.

---

## 2. 프로젝트 목표

본 프로젝트의 핵심 목표는 다음과 같습니다.

- IO 맵 및 사양서 기반 입력 처리 흐름 구성
- 비정형 데이터의 구조화 및 표준화
- PLC 로직 생성을 위한 내부 데이터 모델 구성
- 검증 가능한 코드 생성 파이프라인 구축
- 재사용 가능한 Structured Text 초안 생성 구조 구현

---

## 3. 입력 및 출력

### 입력
- CSV / JSON 형태의 IO 데이터
- 텍스트 기반 설비 사양 정보

### 출력
- IEC 61131-3 Structured Text(ST) 초안
- 중간 검증용 데이터 구조
- 테스트 및 확인용 결과 데이터

---

## 4. 처리 흐름

본 프로젝트는 아래와 같은 파이프라인을 기반으로 구성됩니다.

```text
Input Data
   ↓
Parsing
   ↓
Normalization
   ↓
Intermediate Data Structure
   ↓
Validation
   ↓
ST Draft Generation
