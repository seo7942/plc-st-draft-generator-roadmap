폐수 처리 쪽 자동화 하나 잡아주세요.

탱크가 여러 개 있고 펌프, 밸브, 약품 투입, 교반기, 슬러지 배출, 컨베이어 조금 있습니다.
지금은 수동 조작이 많아서 작업자가 계속 붙어 있어야 합니다.
자동 운전으로 바꾸고 HMI에서 상태 보고 수동도 가능하게 해주세요.

대충 구성은 아래 정도입니다.

- 원수 유입 탱크
- 약품 투입 탱크
- 중화 탱크
- 세정/린스 탱크
- 슬러지 배출 라인
- 최종 배출 라인
- 보조 컨베이어
- 공압 실린더 몇 개
- 송풍기/배기팬
- 히터 일부 있음
- pH, 온도, 유량, 압력, 수위 정도는 봐야 함

펌프랑 밸브가 많습니다.
정확한 수량은 현장 정리 필요합니다.
일단 IO는 예비 포함 500점 정도로 잡아주세요.

LS XGT 기준이면 좋습니다.
나중에 XG5000에 넣을 수 있게 주소표랑 ST/Ladder 초안 나오면 좋겠습니다.
도면도 검토용으로 뽑아주세요.
HMI는 운전 화면, 알람 화면, IO 모니터 정도면 됩니다.

안전 쪽은 꼭 넣어주세요.

- 비상정지
- 안전문
- 누액 감지
- 탱크 저수위
- 탱크 고수위
- 펌프 공회전 방지
- 모터 과부하
- 공압 저하
- 히터 과온
- pH 이상
- 유량 없음
- 밸브 열림/닫힘 시간초과
- 컨베이어 잼
- 스크류 막힘

자동 운전 순서는 대략 이렇게 보면 됩니다.

유입 → 약품 투입 → 교반 → 중화 → 세정 → 린스 → 슬러지 배출 → 최종 배출

각 단계별 타임아웃 넣어주세요.
알람은 래치 방식이면 좋고 Reset 누르기 전까지 유지되게 해주세요.
수동 운전에서도 안전 인터락은 우회되면 안 됩니다.
Spare IO는 실제 로직에 쓰지 말고 예비로만 표시해주세요.

산출물은 아래 정도 필요합니다.

- PLC Package JSON
- IO List
- ST Draft
- Ladder Draft
- HMI 화면 후보
- 전장 도면 후보
- 알람표
- 인터락표
- 타이머/카운터표

실제 현장 투입 확정본은 아니고 검토용입니다.
모든 결과물은 REVIEW ONLY로 표시해주세요.
confirmed_for_plc_run=false 유지해주세요.
